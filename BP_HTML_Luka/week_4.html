Figma Mid-Fidelity Verfijning

Deze week werd besteed aan de Mid-Fidelity fase in Figma. Dit hield in dat alle schermen werden voltooid, de componentenbibliotheek werd gestabiliseerd en de interacties (prototyping) werden toegevoegd. Het mid-fi prototype dient nu als de blauwdruk voor de definitieve High-Fidelity versie.

Introductie tot Git, Github en Netlify

We kregen uitleg over twee cruciale tools voor de moderne ontwikkelaar:

  • **Github:** Uitleg over versiebeheer met Git en hoe we onze code kunnen opslaan, delen en samenwerken via Github repositories.
  • **Netlify:** Demonstratie van hoe we een live website kunnen maken via Continuous Deployment, door onze Github repository te koppelen aan Netlify.
